# 📂 Data Directory

This folder holds the raw and processed datasets used for training.

---

## 📥 How to Download the Dataset

### Option 1: Kaggle CLI (Recommended)

1. Install the Kaggle CLI:
   ```bash
   pip install kaggle
   ```

2. Place your `kaggle.json` API key in `~/.kaggle/kaggle.json`
   (Download it from: https://www.kaggle.com/settings → API → Create New Token)

3. Download and unzip the dataset:
   ```bash
   kaggle datasets download -d clmentbisaillon/fake-and-real-news-dataset
   unzip fake-and-real-news-dataset.zip -d data/raw/
   ```

### Option 2: Manual Download

1. Go to: https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset
2. Click **Download**
3. Unzip the files into `data/raw/`

## 📊 Dataset Overview

| File      | Label | Count  | Source            |
|-----------|-------|--------|-------------------|
| True.csv  | REAL  | 21,417 | Reuters           |
| Fake.csv  | FAKE  | 23,481 | PolitiFact + others |

### Columns in each CSV:
- `title`   — Headline of the article
- `text`    — Full body of the article
- `subject` — Category (politics, world news, etc.)
- `date`    — Publication date
